Microgrid-Manager
February 2020   -   Find on Github
Optimisation
Data-Science
This project is an academic project carried out during the second semester of 2019-2020. The goal is to manage a battery in a microgrid fitted with a solar pannel. This project uses the command predictive control. At every step:
  1. we predict the net demand (difference between the demand and the production)
  2. we set the trajectory battery load in order to minimize the total cost.
  3. only the first value of the prescription is retained
