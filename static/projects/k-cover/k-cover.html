This project is an academic project carried out in October 2021. The goal is to implement an heuritic to cover a graph with a minimum of sensors.
  - Each node has to be covered by at least k sensors
  - Sensors have to be connected together
  - Sensors have a covering radius and a communication radius After a Mixed-interger-Linear formulation of the problem, we implemented a genetic algorithm and a local search (inspired by this article)
